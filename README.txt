Практический проект.

Описание:
Представляет из себя Rest-приложение(Jersey+Spring), с подключённой базой данных PostgreSQL. Отображение в браузере выполнено через JSP (с использованием JSTL). Также реализован пример использования Web-сервлета.
Через JSON можно обратиться к базе и получить ответ так же в формате JSON (использовал расширение POSTMAN для Chrome).

В работе использовал: Sring Data Jpa, Spring Web, Jersey-Spring, JUnit, Java EE (вариант с отображением через сервлет), JSP+JSTL.
Настройка спринга через xml-файл.
В файле order.sql находятся SQL-команды создания таблиц и внесения в них значений.
